# Installation guides

### Requirement

* maven
* docker-compose 3
* java < 12 on JAVA_PATH

### Install

* Run `docker-compose up`
* Install project's artifact with Maven
* Run application from file src/main/java/com/isd/ict/capstoneproject/HelloApplication.java
* Run project `mvn clean javafx:run`
